# Vibe Design System — Grandbook (arsif lokal)

Sumber resmi: [vibe.monday.com](https://vibe.monday.com/?path=/docs/welcome--docs) (Storybook produksi monday.com).

## Isi arsip

| Berkas | Keterangan |
|--------|------------|
| `index.json` | Indeks Storybook v5 (652 entri: docs + story). |
| `manifest.json` | Daftar halaman yang di-scrape: URL iframe, berkas Markdown, jumlah karakter. |
| `pages/*.md` | Teks yang diekstrak per halaman (frontmatter + isi). |
| `mcp-tokens.json` | 912 token @vibe/style dengan contoh pemakaian CSS (dari MCP). |
| `mcp-icons.json` | Katalog ikon @vibe/icons (dari MCP). |
| `scrape-vibe.mjs` | Skrip Playwright untuk mengulang scraping. |

## Statistik

- Entri di `index.json`: **652**
- Halaman terekstrak ke `pages/`: **652**
- Halaman dengan teks kosong (`chars === 0`, biasanya pratinjau visual tanpa label teks di root): **68**
- Gagal otomatis (lalu diperbaiki manual bila ada): **0**

## Metode scraping

1. Mengunduh `/index.json` dari host Storybook.
2. Untuk setiap entri, membuka `iframe.html?id=…&viewMode=docs|story` dengan Playwright (Chrome sistem).
3. Mengekstrak teks dari `.sbdocs-wrapper` (dokumentasi) atau `#storybook-root` (story).
4. Satu halaman docs (`components-search--docs`) tidak merender konten di iframe; diganti dengan ringkasan dari MCP Vibe.
