# Ecosystra (Shadboard layout)

Aplikasi produk berada di **`full-kit/`** — Next.js 15, NextAuth, Ecosystra + Email + Chat + Calendar.

Dari root monorepo (`Ecosystra/`):

- `pnpm dev:3002` — dev server (port 3002)
- `pnpm build` — build produksi

Lihat **`full-kit/README.md`** untuk variabel lingkungan dan skrip database.
